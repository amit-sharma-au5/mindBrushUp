Listing all files and folders
> **ls**

Present working Directory
> **pwd**

Creating Directory
> **mkdir dir_name**

Creating File 
> **touch file_name**

Copy a file 
> **cp from_file to_file**

Remove File
> **rm file_name**

Cut and Paste file Or move a file
>  **mv from_file to_file**

Remove Folder
> **rm -r folder_name** 
> Here -r for recurrsive
